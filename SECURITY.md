# Política de Segurança

A segurança e integridade dos dados dos alunos e professores da FCTE UNB é nossa prioridade.

## Versões Suportadas

Apenas a versão mais recente na branch `master` recebe suporte ativo para correções de segurança.

## Relato de Vulnerabilidades

Se você descobrir uma vulnerabilidade de segurança (como falhas no script de automação, injeção de comandos, ou vazamento de dados confidenciais), **NÃO** abra uma Issue pública.

Por favor, reporte a vulnerabilidade de forma privada usando o recurso **GitHub Security Advisories** na aba de Segurança do repositório, ou entre em contato diretamente com os administradores.

Faremos o possível para responder e corrigir o problema o mais rápido possível.
