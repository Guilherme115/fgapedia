# WikiProvas FCTE UNB 📚

A **WikiProvas FCTE UNB** é um repositório colaborativo ("Wikipedia de Provas") focado em ajudar os estudantes da Faculdade de Ciências e Tecnologias em Engenharia (FCTE) da Universidade de Brasília (UNB). 

Nosso objetivo é democratizar o acesso a materiais de estudo, permitindo que alunos compartilhem e acessem provas antigas, gabaritos e listas de exercícios de semestres anteriores, facilitando a preparação para as avaliações.

## 🚀 Como Contribuir
Se você tem provas antigas guardadas, **você pode ajudar milhares de alunos!** O processo é 100% automático e você não precisa saber programar:
1. Vá na aba de [Issues](../../issues) do repositório.
2. Clique em **New Issue** e selecione "Enviar Nova Prova".
3. Preencha os campos e anexe o seu PDF. 
4. Nosso robô fará o resto!

Para instruções detalhadas ou caso você seja um desenvolvedor querendo contribuir com o código, leia nosso [Guia de Contribuição (CONTRIBUTING.md)](CONTRIBUTING.md).

## 🛠️ Tecnologias
Este projeto opera através de uma arquitetura **Serverless / GitOps**:
- **Frontend**: React + Vite (Hospedado no GitHub Pages).
- **Backend/Ingestão**: GitHub Actions (IssueOps) + Node.js (Com compressão inteligente usando Ghostscript).
- **Banco de Dados**: JSON estático hiper-otimizado.

## ⚖️ Direitos Autorais e Remoção de Conteúdo (Takedown Notice)

Todo o conteúdo deste repositório é enviado colaborativamente pelos próprios estudantes. Nosso objetivo é estritamente educacional e sem fins lucrativos.

Em conformidade com a **Lei de Direitos Autorais Brasileira (Lei nº 9.610/1998)**, nós respeitamos a propriedade intelectual dos professores e criadores de conteúdo. 

**Se você é professor(a) ou autor(a) de algum dos PDFs inseridos aqui e não deseja que seu material seja compartilhado**, por favor, solicite a retirada do material enviando um email para:
📩 **AINDA SEM EMAIL**

Assim que recebermos a solicitação, o documento será **excluído o mais breve possível** do nosso banco de dados e repositório.
