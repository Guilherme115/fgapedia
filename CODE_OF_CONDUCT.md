# Código de Conduta

Como contribuidores e mantenedores deste projeto, comprometemo-nos a respeitar todas as pessoas que contribuem através de issues, pull requests, comentários e qualquer outro meio de comunicação do projeto.

## Nossos Padrões

Exemplos de comportamento que contribuem para criar um ambiente positivo incluem:
* O uso de linguagem acolhedora e inclusiva.
* O respeito a diferentes pontos de vista e experiências.
* A aceitação de críticas construtivas de forma graciosa.
* O foco no que é melhor para a comunidade de estudantes da FCTE UNB.

Exemplos de comportamento inaceitável incluem:
* O uso de linguagem sexualizada ou imagens ofensivas.
* Ataques pessoais, insultos ou comentários depreciativos.
* Assédio público ou privado.
* Compartilhamento de informações confidenciais ou privadas sem consentimento (ex: provas com gabaritos restritos ou dados pessoais de professores não públicos).

## Relato de Problemas
Qualquer violação deste Código de Conduta pode ser reportada abrindo uma Issue confidencial (ou contatando os administradores do repositório diretamente). Todas as queixas serão revisadas e investigadas de forma justa.
