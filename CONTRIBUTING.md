# Contribuindo para a WikiProvas FCTE UNB

Primeiramente, muito obrigado por querer contribuir! Este repositório é feito de alunos para alunos, e a sua ajuda é fundamental.

## Como enviar uma prova antiga?

Nós simplificamos o processo para que você não precise saber programar para ajudar:

1. Vá na aba **Issues** do repositório.
2. Clique no botão verde **New Issue**.
3. Selecione o template **Enviar Nova Prova**.
4. Preencha os campos obrigatórios (Professor, Matéria, Semestre, etc).
5. Arraste e solte o arquivo **PDF** da sua prova na caixa designada.
6. Clique em **Submit new issue**.

Pronto! Nosso robô (GitHub Actions) fará o download do seu PDF, vai comprimi-lo automaticamente e atualizará nosso banco de dados. Você receberá uma notificação na própria Issue assim que ela for processada.

## Como contribuir com o Código (Frontend/Backend)?

Se você é desenvolvedor e quer melhorar a interface ou a automação:
1. Faça um Fork do repositório.
2. Crie uma branch para sua feature (`git checkout -b feature/minha-feature`).
3. Rode `npm install` e `npm run dev` para testar localmente.
4. Faça o commit das suas mudanças (`git commit -m 'feat: adicionei nova funcionalidade'`).
5. Dê Push para a branch (`git push origin feature/minha-feature`).
6. Abra um **Pull Request**.

Qualquer dúvida, abra uma Issue para discutirmos!
